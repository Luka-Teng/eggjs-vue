# project

eggjs + vuejs project

### Frontend DEV

```bash
$ cd /rootDir/project-frontend
$ npm i
$ npm run dev
$ open http://localhost:8080/
```

### Backend DEV

```bash
$ cd /rootDir/project-backend
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### Intro.

```bash
frontend: vue + vuex + vue-router + axios
backend: eggjs + egg-sequelize
```
